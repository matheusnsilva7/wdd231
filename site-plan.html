Sports Explorer – Website Project Plan

This document outlines the planning details for my final website project for WDD 231.

Site Name

Sports Explorer

This name was selected because the website helps users explore a wide variety of sports, discover new activities, and learn interesting details about each sport.

Site Purpose

The purpose of Sports Explorer is to promote an active lifestyle by helping visitors learn about different kinds of sports. The website will provide detailed information fetched dynamically from TheSportsDB API, including sport names, descriptions, and images. It will also include a contact page where users can share their favorite sport.

Scenarios

  • “What new sport can I try and what equipment do I need?”
  • “Where can I learn more details or history about a sport I’m interested in?”

Color Scheme

These colors will be used consistently across the website.

  • Primary Blue (#0b6efd): Headings, navigation highlights, buttons.
  • Secondary Yellow (#ffb703): Accents, icons, borders, call-to-action elements.
  • Dark Gray (#1a1a1a): Main text.
  • Light Gray (#f7f7f7): Background color.

Typography

  • Montserrat: Used for headings (H1–H3) to create a clean visual hierarchy.
  • Open Sans: Used for body text, paragraphs, and navigation links.
